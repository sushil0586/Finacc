# Bank Reconciliation CSV Samples

Use these files to test the import preview, column mapping, saved import profiles, and reconciliation flow.

Recommended test order:
1. Upload the file.
2. Review preview headers and sample rows.
3. Map columns to normalized fields.
4. Save an import profile.
5. Import the file into a session.
